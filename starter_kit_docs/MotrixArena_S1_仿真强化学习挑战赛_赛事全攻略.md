# LEARN RL & WIN ROBOTS

## MotrixArena S1 挑战赛

### 核心亮点：双阶段闯关

#### PHASE_01 // STAGE 1：平地导航 (Flat Ground Navigation)
- /starter_kit/navigation1/
- 四足狗从起点出发，在平地场景中到达目标点。
- 重点考察行走过程中的稳定性。

#### PHASE_02 // STAGE 2：四足越障 (Quadruped Obstacle Course)
- /starter_kit/navigation2/
- 途经坑洼、斜坡、波型地形、楼梯及随机地形。
- 展现算法优化能力，挑战硬核全地形控制。

### 提交材料清单

| 编号 | 材料名称 | 具体要求 |
