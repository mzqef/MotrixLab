# MotrixArena S1 计分规则讲解

## 平地导航 /starter_kit/navigation1/

### 基本规则
机器狗从最外围的蓝色电子围栏和彩灯构筑的圆环出发，初始位置随机生成，依次到达内部的蓝色围栏，最终达到悬浮灯笼下方的圆心。过程中没有出现摔倒和越界行为，最高可得 **20分**。

### 加分点
1. 机器狗到达悬浮灯笼下的电子围栏（蓝色围栏） +1分  
2. 机器狗到达悬浮灯笼下圆心 +1分  
3. 同时在随机起始点区域生成10只机器狗，每只狗最多可得2分，总计最多可得20分

### 扣分点
1. 出发点位置不随机 / 超出边界 / 摔倒等行为 → 扣除该机器狗所有得分  
2. 扣分无连带责任，其余机器狗得分不受影响

## 越障导航 —— Section1 /starter_kit/navigation2

### 基本规则
机器狗从 "START" 平地处出发，初始点位置随机分布在 "START" 平台区域，穿过各个路段到达（身体任何部位进入即视为到达）"2026" 平台处，过程中无摔倒和越界行为，最高可得 **20分**。

### 加分点
1. 坑洼地形的趣味得分区有三个笑脸区域：
   - 进入任意一个笑脸 +4分  
   - 多次进入同一个笑脸区域只计第一次得分  
   - 最高获得 **12分**
2. 斜坡地形的趣味得分区有三个悬浮在“GO”字样上的祝福红包：
   - 到达任意一个红包位置 +2分  
   - 多次进入同一个红包区域只计第一次得分  
   - 最高获得 **6分**
3. 在 "2026" 平台处做出庆祝动作 +2分  
   - 需自定义优雅且明显的庆祝动作

### 扣分点
- 出发点位置不随机 / 超出边界 / 摔倒行为 → 扣除本 Section 所有得分

---

## 越障导航 —— Section2 /starter_kit/navigation2

### 基本规则
机器狗从 "2026" 平台处出发，初始点位置随机分布在 "2026" 平台处，穿过各个路段到达（身体任何部位进入即视为到达）"丙午大吉" 平台处，过程中无摔倒和越界行为，最多可得 **60分**。

### 加分点

1. 通过“2026”平台前的波浪地形到达楼梯 +10分
2. 从左边楼梯到达吊桥 或 从右边楼梯到达河床 +5分
3. 经过吊桥途径拜年红包到达楼梯口 +10分
4. 从楼梯口下来到达"丙午大吉"平台处 +5分
5. 在"丙午大吉"平台处做出优雅且明显的庆祝动作 +5分
6. 经过河床到达楼梯 +5分
7. 获得河床圆形石头上的贺礼红包 +3分 × 5 = 最多15分
8. 获得桥底下的拜年红包 +5分 x 2 = 最多10分

### 扣分点
- 出发点位置不随机 / 超出边界 / 摔倒行为 → 扣除本 Section 所有得分

## 越障导航 —— Section3 /starter_kit/navigation2

### 基本规则
机器狗从 "丙午大吉" 平台处出发，初始点位置随机分布在 "丙午大吉" 平台处，穿过各个路段到达（身体任何部位进入即视为到达）"中国结" 平台处，过程中无摔倒和越界行为，最多可得 **25分**。

### 加分点
1. 穿过“丙午大吉”平台前方的滚动球区域并且到达随机地形，分为两种情况：
   - 没有碰到滚球直接到达随机地形 → +10分  
   - 碰到滚球且能不摔倒、不出界到达随机地形 → +15分  
2. 通过随机地形到达"中国结"平台处 → +5分  
3. 在“中国结”平台处做出庆祝动作 → +5分  

### 扣分点
- 出发点位置不随机 / 超出边界 / 摔倒行为 → 扣除本 Section 所有得分

---

## 最终排名计算方式

1. 按总分数（平地导航分数 + 越障导航赛道最终得分）来排名，分数越高排名越前
2. 总分数相同，则按越障导航赛道总时长的时间来排名，时间越短排名越前
3. 若使用单个模型跑完越障导航赛道全程，到达终点位置——"中国结"平台，过程中无“摔倒”和“越界”情况，则在同分数段无论时间长短均排名第一
4. 若多个选手满足第3条条件且分数一致，则按时间排名，时间越短排名越前
